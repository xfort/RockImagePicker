# RockImagePicker 
[![](https://jitpack.io/v/xfort/RockImagePicker.svg)](https://jitpack.io/#xfort/RockImagePicker)

Android 图片选择库
1. 拍照、图片选取
2. 可设置最大选择图片数
3. 可设置已选中的图片
4. 适配android 11

#Download

使用jitpack
```
	allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
```
```
dependencies {
	        implementation 'com.github.xfort:RockImagePicker:1.1'
	}
```
